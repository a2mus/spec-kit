# Quickstart: Clean Fork for Upstream Sync

**Feature Branch**: `002-clean-runtime-artifacts` | **Date**: 2026-06-22

## Prerequisites

- Git 2.x+
- PowerShell 7+ (Windows) or Bash (Linux/macOS)
- Python 3.11+ with [uv](https://docs.astral.sh/uv/) installed
- The spec-kit CLI (`specify`) installed via `uv tool install`

## Implementation Steps (Operator Guide)

### Step 1: Create the Restore Point

Before any changes, tag the current state:

```powershell
git tag pre-cleanup-002 -m "Restore point before runtime artifact cleanup"
git push origin pre-cleanup-002
```

### Step 2: Update `.gitignore`

Add the fork-specific runtime artifact rules. See the plan's `.gitignore`
changes section for the exact rules. Key additions:

```gitignore
# Fork-specific: runtime artifacts (regenerated by `specify init`)
.agents/
.claude/
.kilocode/
.opencode/
CLAUDE.md

# .specify/ runtime subdirectories
.specify/scripts/
.specify/templates/
.specify/memory/
.specify/integrations/
.specify/workflows/
.specify/extensions/
.specify/feature.json

# Keep project config files tracked (negation rules)
!.specify/extensions.yml
!.specify/init-options.json
!.specify/integration.json
```

### Step 3: Remove Files from Git Index

```powershell
# Agent integration directories
git rm -r --cached .agents/
git rm -r --cached .claude/

# .specify/ runtime subdirectories
git rm -r --cached .specify/scripts/
git rm -r --cached .specify/templates/
git rm -r --cached .specify/memory/
git rm -r --cached .specify/integrations/
git rm -r --cached .specify/workflows/
git rm -r --cached .specify/extensions/

# Generated context file
git rm --cached CLAUDE.md
```

### Step 4: Verify Local Files Are Intact

```powershell
# All files should still exist on disk
Test-Path .agents/skills/api-design/SKILL.md  # Should be True
Test-Path .claude/skills/api-design/SKILL.md  # Should be True
Test-Path .specify/scripts/powershell/common.ps1  # Should be True
```

### Step 5: Verify Git Tracking

```powershell
# These should return empty
git ls-files .agents/ .claude/ .specify/scripts/ .specify/templates/

# These should return the three config files
git ls-files .specify/extensions.yml .specify/init-options.json .specify/integration.json

# Ponytail should be intact
git ls-files extensions/ponytail/
```

### Step 6: Commit and Push

```powershell
git add .gitignore
git commit -m "chore: untrack runtime artifacts for clean upstream sync"
git push origin 002-clean-runtime-artifacts
```

### Step 7: Test Upstream Merge Compatibility

```powershell
git fetch upstream main
git merge upstream/main --no-commit --no-ff
# Check for conflicts on previously-tracked runtime paths
git diff --name-only --diff-filter=U
# Should show zero conflicts on .agents/, .claude/, .specify/ runtime paths
git merge --abort  # or commit if clean
```

## Fresh Clone Setup (Developer Guide)

After the cleanup, a fresh clone needs to regenerate runtime artifacts:

```powershell
git clone https://github.com/a2mus/spec-kit.git
cd spec-kit

# Install the spec-kit CLI from this fork
uv tool install specify-cli --from git+https://github.com/a2mus/spec-kit.git@main

# Regenerate runtime artifacts
specify init --here --integration opencode --ignore-agent-tools --force

# Verify
specify --version
```

## Rollback

If anything goes wrong, restore the pre-cleanup state:

```powershell
# Full rollback to pre-cleanup tracked-file set
git revert <cleanup-commit-sha>

# Or checkout the tagged state
git checkout pre-cleanup-002 -- .
git checkout HEAD -- .gitignore  # optionally keep new ignore rules
```
