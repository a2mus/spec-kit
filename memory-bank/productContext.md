# Product Context

## Purpose
Provide a practical tool to define virtual fences, observe cattle movement, and track basic health metrics, enabling non-invasive monitoring and management.

## Problems Solved
- Reduces manual effort to monitor herd locations.
- Offers near real-time telemetry for decision making.
- Centralizes data for analysis and historical review.

## Key Experiences
- Map-based UI for viewing and drawing fences.
- Near real-time latest-collar updates.
- Simple flows to create and save geofences.
- Clear feedback on simulator/ingestion status.

## UX Principles
- Minimal friction: straightforward map controls.
- Fast feedback: latest telemetry endpoint updates.
- Reliability: data durability via TimescaleDB.
